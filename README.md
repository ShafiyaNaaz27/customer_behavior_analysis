# customer_behavior_analysis
# 📊 Data Analytics Project

## 📌 Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data and ending with actionable business insights. The project includes data cleaning, Exploratory Data Analysis (EDA), SQL-based analysis, interactive dashboard creation in Power BI, and business reporting.

The objective is to transform raw data into meaningful insights that support data-driven decision-making.

---

## 📂 Dataset

The dataset contains business-related information such as customers, products, sales, orders, and transactions.

The data was analyzed to identify:
- Sales trends
- Customer behavior
- Product performance
- Business growth opportunities
- Key performance metrics

## 🔄 Project Workflow

### 1. Data Loading
- Imported dataset into Python
- Inspected data structure
- Verified data types and missing values

### 2. Data Cleaning
- Removed duplicate records
- Handled missing values
- Corrected inconsistent data
- Converted data types
- Created new calculated columns where required

### 3. Exploratory Data Analysis (EDA)
- Performed statistical analysis
- Identified trends and patterns
- Visualized distributions and relationships
- Analyzed customer and sales behavior

### 4. SQL Analysis
Executed SQL queries to answer business questions using:
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- JOINs
- Aggregate Functions
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions

Database platforms used:
- PostgreSQL
- MySQL
- SQL Server

### 5. Dashboard Development
Built an interactive Power BI dashboard featuring:
- KPI Cards
- Sales Trends
- Customer Insights
- Product Performance
- Regional Analysis
- Interactive Filters and Slicers

### 6. Business Report
Prepared a report summarizing:
- Key findings
- Business insights
- Recommendations
- Performance metrics

### 7. Presentation
Created a professional project presentation using **Gamma AI** to communicate the project's objectives, methodology, insights, and recommendations.

## ▶️ How to Run

1. Clone this repository.

```bash
git clone https://github.com/your-username/project-name.git
```

2. Install the required Python libraries.

```bash
pip install -r requirements.txt
```

3. Open the Jupyter Notebook.

```bash
jupyter notebook
```

4. Run the notebook to perform data cleaning and analysis.

5. Execute the SQL queries using PostgreSQL, MySQL, or SQL Server.

6. Open the Power BI (`.pbix`) file to explore the interactive dashboard.

7. Review the business report and presentation for project insights.
